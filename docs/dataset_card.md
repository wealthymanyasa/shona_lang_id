# English–Shona Language ID Dataset

This dataset contains 763,546 English–Shona sentence pairs, cleaned and balanced for language identification tasks.

## Structure
- `shona_en_train.txt` – training set
- `shona_en_validation.txt` – validation set
- `shona_en_test.txt` – test set
